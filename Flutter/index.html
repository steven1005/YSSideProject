        hello_world/
        ├── lib/

2.理解 main.dart

打開 lib/main.dart，逐行解釋：

        import 'package:flutter/material.dart';  // ← 導入 Flutter UI 組件

        void main() {
        runApp(const MyApp());  // ← 應用程式的入口點，運行 MyApp
        }

概念：Widget

Flutter 中一切都是 Widget（組件）

Widget 是構建 UI 的基本單位

        class MyApp extends StatelessWidget {
        const MyApp({super.key});

        @override
        Widget build(BuildContext context) {
            return MaterialApp(
            title: 'Hello World',
            theme: ThemeData(primarySwatch: Colors.blue),  // ← 藍色主題
            home: const HelloWorldPage(),  // ← 首頁
            );
        }
        }

StatelessWidget：無狀態（不變化的 UI）

MaterialApp：包裝了應用的主框架

home：定義應用打開時顯示的頁面

        class HelloWorldPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Hello World')),  // ← 頂部欄
      body: const Center(
        child: Text(
          'Hello, World!',
          style: TextStyle(fontSize: 48, fontWeight: FontWeight.bold),
        ),
      ),
    );
  }
}

Scaffold：提供 Material Design 的佈局（appBar + body）

Center：把內容居中

Text：文本組件

重要概念：StatelessWidget vs StatefulWidget

StatelessWidget StatefulWidget
用途 靜態 UI（不變化） 動態 UI（會改變）
例子 文本、圖片、按鈕 計數器、表單輸入

加入一個計數按鈕

試試替換 HelloWorldPage 為 StatefulWidget：

        class HelloWorldPage extends StatefulWidget {
  const HelloWorldPage({super.key});

  @override
  State createState() => _HelloWorldPageState();
}

class _HelloWorldPageState extends State {
  int counter = 0;  // ← 狀態（會變化）

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Hello World')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text(
              'Hello, World!',
              style: TextStyle(fontSize: 48, fontWeight: FontWeight.bold),
            ),
            const SizedBox(height: 20),
            Text('計數: $counter', style: const TextStyle(fontSize: 24)),
            const SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                setState(() {
                  counter++;  // ← 更新狀態，自動重繪 UI
                });
              },
              child: const Text('點擊'),
            ),
          ],
        ),
      ),
    );
  }
}

關鍵點：

setState(() { ... })：告訴 Flutter「我改變了狀態，請重新繪製」
