jQuery 教學

本教學旨在為初學者提供一個全面而詳細的 jQuery 入門指引，並逐步深入到更進階的應用。

常用 jQuery 方法總覽

大項目 功能 (方法) 描述

1: 什麼是 jQuery？ - 快速入門與引入

jQuery 是一個快速、小巧且功能豐富的 JavaScript 函式庫。它使得 HTML 文件遍歷和操作、事件處理、動畫和 Ajax 等操作變得更加簡單。它的設計理念是 "write less, do more"（寫得更少，做得更多）。要使用它，您需要先在您的 HTML 頁面中引入 jQuery 檔案，通常是透過 CDN 或下載到本地。

HTML 範例：引入 jQuery

<!-- 使用 Google CDN 引入 jQuery -->

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

2: 核心語法與強大選擇器 - 精準定位元素

jQuery 的核心語法是 $(selector).action()。$ 符號是 jQuery 的別名，selector 用來查詢 HTML 元素，而 action() 則是對選中的元素執行的操作。選擇器是 jQuery 最強大的部分之一，它基於 CSS 選擇器，讓您可以非常精準地定位到頁面上的任何元素。

HTML 範例：多種選擇器應用

<div id="container">
        <p class="text-item">這是一個段落。</p>
        <span class="text-item">這是一個 span。</span>
        <button>點擊我</button>
    </div>
    <p>另一個段落。</p>

jQuery 範例：選擇器實戰

// 元素選擇器: 選取所有 <p> 元素，並改變其顏色
$("p").css("color", "blue");

// ID 選擇器: 選取 id="container" 的元素，並添加邊框
$("#container").css("border", "1px solid red");

// Class 選擇器: 選取所有 class="text-item" 的元素，並加粗字體
$(".text-item").css("font-weight", "bold");

// 組合選擇器: 選取 #container 內的所有 <p> 元素
$("#container p").css("background-color", "yellow");

// 屬性選擇器: 選取所有帶有 'class' 屬性的元素
$([class]).css("padding", "5px");

3: 事件處理 - 響應使用者互動

事件是使用者在網頁上的行為，例如點擊、滑鼠移入、鍵盤輸入等。jQuery 提供了簡潔且跨瀏覽器兼容的方式來處理這些事件。.on() 是最推薦的事件綁定方法，它功能強大且靈活。

HTML 範例：事件綁定

<button id="myButton">點擊我 (.on)</button>
<button id="clickButton">點擊我 (.click)</button>
<div id="hoverBox" style="width:150px; height:50px; background-color:lightgray; margin-top: 10px;">滑鼠移入這裡 (.hover)</div>
<input type="text" id="myInput" placeholder="在這裡輸入文字">

jQuery 範例：事件處理實戰

// 為按鈕綁定點擊事件
$("#myButton").on("click", function() {
  alert("按鈕被點擊了！");
});

// 為 div 綁定滑鼠移入/移出事件
$("#hoverBox").on({
  mouseenter: function() {
    $(this).css("background-color", "yellow");
  },
  mouseleave: function() {
    $(this).css("background-color", "lightgray");
  }
});

// 為輸入框綁定鍵盤抬起事件
$("#myInput").on("keyup", function() {
  console.log("輸入框內容: " + $(this).val());
});

4: 動態特效與動畫 - 讓網頁生動起來

jQuery 內建了多種常用特效，讓您可以輕易地為網頁加入動畫，提升使用者體驗。這些特效通常都支援速度參數（如 'slow', 'fast', 或毫秒數）和回調函數。

HTML 範例：動畫元素

<button id="toggleBtn">切換顯示/隱藏</button>
<div id="myBox" style="width:150px; height:150px; background-color:orange; margin-top:10px;"></div>

<button id="fadeBtn">淡入/淡出</button>
<div id="fader" style="width:150px; height:150px; background-color:purple; margin-top:10px;"></div>

<button id="slideBtn">滑動切換</button>
<div id="slider" style="width:150px; height:150px; background-color:teal; margin-top:10px; overflow:hidden;">滑動內容</div>

jQuery 範例：特效實戰

// 隱藏和顯示
$("#toggleBtn").on("click", function() {
  $("#myBox").toggle("slow"); // 慢速切換顯示/隱藏
});

// 淡入和淡出
$("#fadeBtn").on("click", function() {
  $("#fader").fadeToggle(800); // 800毫秒淡入/淡出
});

// 滑動效果
$("#slideBtn").on("click", function() {
  $("#slider").slideToggle("fast", function() {
    // 動畫完成後的回調函數
    console.log("滑動動畫完成！");
  });
});

// 自定義動畫：使用 .animate()
$("#myBox").on("dblclick", function() {
  $(this).animate({
    width: "200px",
    opacity: "0.5",
    marginLeft: "100px"
  }, 1500, function() {
    alert("自定義動畫完成！");
  });
});

5: DOM 內容與屬性操作 - 讀寫元素資訊

DOM (Document Object Model) 是 HTML 和 XML 文件的程式介面。它將頁面解析為一個樹狀結構，讓程式語言可以存取和修改頁面內容。jQuery 提供了簡潔的方法來操作 DOM 元素的內容和屬性。對於 checked, selected, disabled 這類布林屬性，建議使用 .prop() 方法。

HTML 範例：待操作元素

<p id="myParagraph">這是一段原始文字。</p>
<div id="myDiv">這是 <strong>HTML</strong> 內容。</div>
<input type="text" id="myInput" value="預設值">
<a id="myLink" href="#">點擊這裡</a>
<br><br>
<input type="checkbox" id="myCheckbox"> <label for="myCheckbox">同意條款</label>

jQuery 範例：內容與屬性操作實戰

// .text() - 取得或設定純文字內容
let currentText = $("#myParagraph").text(); // 取得文字
console.log("原始文字: " + currentText);
$("#myParagraph").text("這段文字已被 jQuery 修改。"); // 設定文字

// .html() - 取得或設定 HTML 內容
let currentHtml = $("#myDiv").html(); // 取得 HTML
console.log("原始 HTML: " + currentHtml);
$("#myDiv").html("這是 <em>新的</em> HTML 內容。"); // 設定 HTML

// .val() - 取得或設定表單欄位的值
let inputValue = $("#myInput").val(); // 取得輸入框的值
console.log("輸入框的值: " + inputValue);
$("#myInput").val("新的輸入值"); // 設定輸入框的值

// .attr() - 取得或設定屬性
let linkHref = $("#myLink").attr("href"); // 取得 href 屬性
console.log("連結的 href: " + linkHref);
$("#myLink").attr({
  "href": "https://www.google.com",
  "target": "_blank", // 設定多個屬性
  "title": "前往 Google"
});

// .prop() - 取得或設定屬性 (Properties)
// .prop() 用於布林值的屬性，如 checked, selected, disabled
let isChecked = $("#myCheckbox").prop("checked"); // 取得 checked 狀態 (true/false)
console.log("Checkbox 初始狀態: " + isChecked);
$("#myCheckbox").prop("checked", true); // 勾選 checkbox
console.log("Checkbox 更新後狀態: " + $("#myCheckbox").prop("checked"));

6: DOM 元素的新增、移除與替換

jQuery 提供了多種方法來動態地在文件中新增、移除或替換 HTML 元素，這對於建立動態內容非常有用。

HTML 範例：操作目標

<ul id="myList">
    <li>項目 A</li>
    <li>項目 B</li>
</ul>

<div id="container">
    <p>這是一個段落。</p>
    <span class="removable">這個 span 將被移除。</span>
</div>

<div id="oldElement">舊的元素</div>

jQuery 範例：元素操作實戰

// append() - 在元素內部結尾處插入內容
$("#myList").append("<li>新項目 C</li>");

// prepend() - 在元素內部開頭處插入內容
$("#myList").prepend("<li>最前面的項目</li>");

// after() - 在元素外部後面插入內容
$("#myList").after("<p>這是列表後面的新段落。</p>");

// before() - 在元素外部前面插入內容
$("#myList").before("<h2>這是列表前面的新標題。</h2>");

// remove() - 移除選中的元素及其所有子元素
$(".removable").remove();

// empty() - 清空元素的所有子元素，但保留元素本身
$("#container").empty();

// replaceWith() - 替換選中的元素
$("#oldElement").replaceWith("<div id=\"newElement\">新的元素</div>");

7: CSS 與樣式操作 - 動態控制外觀

jQuery 提供了多種方法來動態地新增、移除、切換或直接設定 CSS 樣式，讓您可以根據使用者互動或資料變化來改變網頁的外觀。

HTML 範例：樣式目標

<div id="myBox" style="width:100px; height:100px; background-color:blue;"></div>
<p class="status-text">目前狀態：正常</p>
<button id="toggleStyleBtn">切換樣式</button>

jQuery 範例：樣式操作實戰

// addClass() - 新增一個或多個 class
$("p").addClass("highlight blue-text");

// removeClass() - 移除 class
$("p").removeClass("blue-text");

// toggleClass() - 切換 class
$("div").on("click", function() {
  $(this).toggleClass("active");
});

// css() - 直接設定 CSS 屬性
$("#box").css({
  "background-color": "red",
  "border-radius": "10px",
  "opacity": "0.8"
});

// 取得 CSS 屬性值
let bgColor = $("#myBox").css("background-color");
console.log("背景顏色: " + bgColor);

8: DOM 遍歷 (Traversing) - 探索元素關係

DOM 遍歷是指在 DOM 樹中移動，從一個選定的元素開始，尋找其父、子、兄弟或其他相關的元素。這對於複雜的頁面操作至關重要。

HTML 範例：複雜結構

<div class="container">
    <ul>
        <li>項目 1</li>
        <li class="active">項目 2</li>
        <li>項目 3</li>
    </ul>
    <p>這是容器內的段落。</p>
    <span>這是容器內的 span。</span>
</div>
<div class="another-container"></div>

jQuery 範例：遍歷實戰

// parent() - 找到直接父元素
$("li.active").parent().css("border", "2px solid blue");

// children() - 找到所有直接子元素
$("ul").children().css("list-style-type", "none");

// find() - 找到後代元素中符合條件的元素
$(".container").find("p").css("font-style", "italic");

// siblings() - 找到所有同層的兄弟元素
$("li.active").siblings().css("opacity", "0.5");

// next() / prev() - 找到緊鄰的下一個/上一個兄弟元素
$("li.active").next().css("text-decoration", "underline");
$("li.active").prev().css("text-decoration", "line-through");

// closest() - 找到最近的符合條件的祖先元素
$("span").closest(".container").css("background-color", "#eee");

9: Chaining (鏈式操作) - 提升程式碼可讀性與效率

鏈式操作是 jQuery 的一個強大特性，它允許您在一個 jQuery 物件上連續呼叫多個方法，而無需重複選取元素。這使得程式碼更簡潔、可讀性更高，並且由於減少了 DOM 查詢次數，也能提升效率。

HTML 範例：單一目標元素

<div id="chainedElement" style="width:100px; height:100px; background-color:gray;"></div>

jQuery 範例：鏈式操作實戰

$("#chainedElement")
  .css({"border": "2px solid blue", "margin": "10px"})
  .slideUp(1000)
  .delay(500)
  .fadeIn(800, function() {
    alert("動畫序列完成！");
  });

10: AJAX 簡介 - 非同步資料載入

AJAX (Asynchronous JavaScript and XML) 是一種在不重新整理整個頁面的情況下，與伺服器交換資料並更新部分網頁的技術。jQuery 的 $.ajax() 方法是處理 AJAX 的主要方式，它簡化了複雜的 XMLHttpRequest 物件操作。

HTML 範例：資料顯示區

<button id="loadDataBtn">載入使用者資料</button>
<div id="userData">資料將顯示在這裡...</div>

jQuery 範例：基本 AJAX 請求

let isShown = false;
  let dataHtml = '';

  $("#loadDataBtn").on("click", function() {
    if (!isShown) {
      // 載入資料
      $.ajax({
        url: "https://jsonplaceholder.typicode.com/users/1", // 範例 API
        method: "GET",
        dataType: "json", // 預期回傳 JSON 格式
        success: function(data) {
          // 請求成功時執行
          dataHtml = `
            <h3>使用者資訊</h3>
            <p>姓名: ${data.name}</p>
            <p>Email: ${data.email}</p>
          `;
          $("#userData").html(dataHtml);
          $("#loadDataBtn").text("隱藏使用者資料");
          isShown = true;
        },
        error: function(xhr, status, error) {
          // 請求失敗時執行
          $("#userData").html(`<p style="color:red;">載入資料失敗: ${status}</p>`);
          console.error("AJAX 錯誤: ", status, error);
          // 錯誤時不改變 isShown，允許重試
        },
        complete: function() {
          // 請求完成 (無論成功或失敗) 時執行
          console.log("AJAX 請求完成。");
        }
      });
    } else {
      // 隱藏資料
      $("#userData").html("資料將顯示在這裡...");
      $("#loadDataBtn").text("載入使用者資料");
      isShown = false;
    }
  });

11: AJAX - 常用簡寫方法與錯誤處理

除了功能完整的 $.ajax()，jQuery 也提供了一些常用的簡寫方法，讓常見的 AJAX 請求更為簡潔。同時，良好的錯誤處理是任何網路請求的關鍵。

HTML 範例：不同請求按鈕

<button id="getBtn">GET 資料</button>
<button id="postBtn">POST 資料</button>
<button id="jsonBtn">GET JSON</button>
<div id="ajaxResult">結果顯示區...</div>

jQuery 範例：簡寫方法與錯誤處理

// $.get() - 執行 GET 請求
$("#getBtn").on("click", function() {
  $.get("https://jsonplaceholder.typicode.com/posts/1", function(data) {
    $("#ajaxResult").html(`<p>GET 成功: ${data.title}</p>`);
  }).fail(function(xhr, status, error) {
    $("#ajaxResult").html(`<p style="color:red;">GET 失敗: ${status}</p>`);
  });
});

// $.post() - 執行 POST 請求
$("#postBtn").on("click", function() {
  $.post("https://jsonplaceholder.typicode.com/posts", { title: 'foo', body: 'bar', userId: 1 }, function(data) {
    $("#ajaxResult").html(`<p>POST 成功: ${data.id} - ${data.title}</p>`);
  }).fail(function(xhr, status, error) {
    $("#ajaxResult").html(`<p style="color:red;">POST 失敗: ${status}</p>`);
  });
});

// $.getJSON() - 執行 GET 請求並預期 JSON 回傳
$("#jsonBtn").on("click", function() {
  $.getJSON("https://jsonplaceholder.typicode.com/users/1", function(data) {
    $("#ajaxResult").html(`<p>JSON 成功: ${data.name}</p>`);
  }).fail(function(xhr, status, error) {
    $("#ajaxResult").html(`<p style="color:red;">JSON 失敗: ${status}</p>`);
  });
});

12: 表單處理 - 收集與提交資料

jQuery 簡化了表單資料的收集、驗證和提交。.serialize() 和 .serializeArray() 是處理表單資料的常用方法。

HTML 範例：註冊表單

<form id="myForm">
    <label for="username">使用者名稱:</label>
    <input type="text" id="username" name="username" required><br><br>
    
    <label for="email">電子郵件:</label>
    <input type="email" id="email" name="email" required><br><br>
    
    <input type="checkbox" id="agree" name="agree" value="yes">
    <label for="agree">我同意條款</label><br><br>
    
    <button type="submit">提交表單</button>
</form>
<div id="formResult"></div>

jQuery 範例：表單資料處理

$("#myForm").on("submit", function(event) {
  event.preventDefault(); // 阻止表單預設提交行為

  // .serialize() - 將表單資料序列化為 URL 編碼的字串
  let formData = $(this).serialize();
  console.log("序列化字串: " + formData);
  $("#formResult").html(`<p>序列化字串: ${formData}</p>`);

  // .serializeArray() - 將表單資料序列化為 JSON 物件陣列
  let formDataArray = $(this).serializeArray();
  console.log("序列化陣列: ", formDataArray);
  $("#formResult").append(`<p>序列化陣列: ${JSON.stringify(formDataArray)}</p>`);

  // 這裡可以將 formData 透過 AJAX 提交到伺服器
  // $.post("/api/submit-form", formData, function(response) { ... });
});

13: 工具方法 (Utility Methods) - 輔助開發

jQuery 提供了一系列實用的工具方法，這些方法不直接操作 DOM，而是提供通用的功能，輔助 JavaScript 開發。

HTML 範例：無特定 HTML，純 JS 邏輯

<div id="output"></div>

jQuery 範例：工具方法實戰

let arr = [1, 2, 3, 4, 5];
let obj = { a: 1, b: 2, c: 3 };
let output = $("#output");

// $.each() - 遍歷陣列或物件
output.append("<h3>$.each() 遍歷:</h3>");
$.each(arr, function(index, value) {
  output.append(`<p>陣列索引 ${index}: ${value}</p>`);
});
$.each(obj, function(key, value) {
  output.append(`<p>物件鍵 ${key}: ${value}</p>`);
});

// $.map() - 遍歷陣列並建立新陣列
let newArr = $.map(arr, function(value, index) {
  return value * 2;
});
output.append(`<p>$.map() 結果: ${newArr.join(", ")}</p>`);

// $.trim() - 移除字串兩端的空白
let str = "  Hello World  ";
output.append(`<p>$.trim() 結果: '${$.trim(str)}'</p>`);

// $.isArray() - 判斷是否為陣列
output.append(`<p>[1,2,3] 是陣列嗎? ${$.isArray([1,2,3])}</p>`);
output.append(`<p>{} 是陣列嗎? ${$.isArray({})}</p>`);

// $.extend() - 合併物件
let obj1 = { name: "Alice", age: 30 };
let obj2 = { city: "New York", age: 31 };
let mergedObj = $.extend({}, obj1, obj2); // 深度複製
output.append(`<p>$.extend() 結果: ${JSON.stringify(mergedObj)}</p>`);

14: 撰寫 jQuery 外掛 (Plugins) - 擴展功能

您可以撰寫自己的 jQuery 外掛來擴展 jQuery 的功能，將重複的程式碼封裝成可重用的模組。這通常是透過擴展 $.fn 物件來實現的。

HTML 範例：外掛目標

<div id="myPluginDiv" style="width:100px; height:100px; background-color:blue;">點擊我</div>

jQuery 範例：自定義外掛

// 定義一個簡單的 jQuery 外掛
$.fn.myHighlighter = function(options) {
  // 預設選項
  let settings = $.extend({
    color: "yellow",
    backgroundColor: "red",
    duration: 500
  }, options);

  return this.each(function() {
    // this 指向當前被選中的 DOM 元素
    let $this = $(this);

    $this.on("click", function() {
      $this.animate({
        backgroundColor: settings.backgroundColor,
        color: settings.color
      }, settings.duration, function() {
        // 動畫完成後恢復
        $this.animate({
          backgroundColor: "",
          color: ""
        }, settings.duration);
      });
    });
  });
};

// 使用自定義外掛
$("#myPluginDiv").myHighlighter({ color: "white", backgroundColor: "green" });

15: Data 屬性 (Data Attributes) - HTML 與 JS 互動

HTML5 引入了 data-* 屬性，允許您在 HTML 元素上儲存自定義資料，而無需使用非標準屬性或 DOM 屬性。jQuery 的 .data() 方法簡化了這些資料的存取。

HTML 範例：帶有 Data 屬性的元素

    產品價格: <span id="productPrice"></span>
</div>
<button id="showProductBtn">顯示產品資訊</button>

jQuery 範例：讀取與設定 Data 屬性

$("#showProductBtn").on("click", function() {
  let $productDiv = $("#product");

  // 讀取 data 屬性
  let productId = $productDiv.data("id");
  let productName = $productDiv.data("name");
  let productPrice = $productDiv.data("price");

  $("#productName").text(productName);
  $("#productPrice").text(`$${productPrice}`);

  console.log(`產品 ID: ${productId}, 名稱: ${productName}, 價格: ${productPrice}`);

  // 設定 data 屬性
  $productDiv.data("stock", 50); // 新增一個 data-stock 屬性
  console.log("產品庫存: " + $productDiv.data("stock"));

  // 也可以一次設定多個
  $productDiv.data({
    "category": "Electronics",
    "available": true
  });
});

16: 事件委派 (Event Delegation) - 處理動態內容

事件委派是一種高效的事件處理模式，特別適用於動態新增的元素或大量相似元素。它透過將事件監聽器附加到父元素，而不是每個子元素來實現。

<ul id="myDynamicList">

<button id="addListItemBtn">新增列表項目</button>

jQuery 範例：事件委派實戰

// 錯誤的綁定方式 (對動態新增的元素無效)
// $("#myDynamicList li").on("click", function() {
//   alert("點擊了: " + $(this).text());
// });

// 正確的事件委派方式
$("#myDynamicList").on("click", "li", function() {
  alert("透過委派點擊了: " + $(this).text());
});

17: Deferred Objects / Promises - 處理非同步操作

jQuery 的 Deferred 物件提供了一種強大的方式來管理非同步操作（如 AJAX 請求、動畫完成等）的狀態和回調。它類似於 ES6 的 Promise。

HTML 範例：非同步操作結果

<button id="fetchDataBtn">獲取資料</button>
<div id="asyncResult">資料載入中...</div>

jQuery 範例：Deferred 物件應用

$("#fetchDataBtn").on("click", function() {
  $("#asyncResult").text("資料載入中...");

  // 模擬一個非同步操作 (例如 AJAX 請求)
  let deferred = $.Deferred();

  setTimeout(function() {
    let success = Math.random() > 0.5; // 隨機成功或失敗
    if (success) {
      deferred.resolve("資料成功載入！"); // 成功時呼叫 resolve
    } else {
      deferred.reject("資料載入失敗！"); // 失敗時呼叫 reject
    }
  }, 2000); // 2秒後完成

  // 處理 Deferred 物件的狀態
  deferred.done(function(message) {
    $("#asyncResult").css("color", "green").text(message);
  }).fail(function(message) {
    $("#asyncResult").css("color", "red").text(message);
  }).always(function() {
    console.log("操作完成。");
  });
});

18: 自定義事件 (Custom Events) - 模組間通訊

除了瀏覽器內建的事件（如 click, mouseover），您也可以建立和觸發自己的自定義事件。這對於模組化開發和組件之間的通訊非常有用。

HTML 範例：事件發送與接收者

<button id="triggerEventBtn">觸發自定義事件</button>
<div id="eventReceiver" style="border:1px solid gray; padding:10px; margin-top:10px;">等待事件...</div>

jQuery 範例：自定義事件實戰

// 綁定自定義事件 'myCustomEvent'
$("#eventReceiver").on("myCustomEvent", function(event, param1, param2) {
  $(this).text(`接收到自定義事件！參數: ${param1}, ${param2}`);
  $(this).css("background-color", "lightgreen");
});

// 觸發自定義事件
$("#triggerEventBtn").on("click", function() {
  // .trigger() 用於觸發事件，可以傳遞額外參數
  $("#eventReceiver").trigger("myCustomEvent", ["Hello", "World"]);
});

// 也可以在觸發時傳遞事件物件
$("#triggerEventBtn").on("dblclick", function() {
  let customEvent = $.Event("anotherCustomEvent");
  customEvent.message = "這是雙擊觸發的事件";
  $("#eventReceiver").trigger(customEvent);
});

$("#eventReceiver").on("anotherCustomEvent", function(event) {
  alert("接收到另一個自定義事件: " + event.message);
});
