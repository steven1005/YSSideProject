1.名詞解釋

  • HTML

    • 全稱：Hypertext Markup Language
    • 內容：比如：文字、圖片、影片等、無法實現特效、排版、特殊版型等
    • 用途：僅用於放置內容，設計與樣式需由CSS實現

  • CSS

    • 全稱：Cascading Style Sheets
    • 樣式與設計：顏色、字型、大小、位置等
    • 排版與定位：用於切版、版型、元素位置的設定

  • JS、jQuery

    • 全稱：JavaScript
    • 行為：互動、動畫、特效等，可以在瀏覽器上執行，簡稱JS
    • 動態變更：可以在使用者動作後，變更CSS樣式，或是改變HTML內容
    • jQuery：是由JavaScript寫出來的應用函式庫，簡化JavaScript的寫法

  • Bootstrap

    • 由HTML、CSS和JavaScript寫成的前端框架
    • 目標：幫助開發者建立美觀、RWD響應式網頁(適應各種螢幕尺寸)
    • 用途：無需自行編寫大量的CSS代碼

  • https://cdnjs.com/

    • 內容傳遞網路(CDN)
    • 目標：用於在靠近終端使用者的位置快取內容
    • 用途：允許快速傳輸載入網際網路內容所需的資產，包括 HTML 頁面、JavaScript 檔案、樣式表、影像和影片

3. HTML 標籤

HTML Tag 用途說明
<title> 瀏覽器的分頁標題
<h1> ~ <h6> 數字越大字越小，用來標示階層式標題
<p> 段落，用於一般文字內容
<br> 斷行，插入換行符號（非段落）
<mark> 標記，用於突出字型
<img src="..."> 圖片，用於顯示圖像
<a href="..."> 超連結，連到其他頁面或網址
<table>、<thead>、<tr>、<td> 表格
<ol>、<ul>、<li> 有排序清單、無排序清單、清單項目

每層都用 <div>會導致結構不清、維護困難，稱為 divitis（div過多症）

語意 tag 用來替代 div 的情境
<header> 頁首區塊
<nav> 導覽列
<main> 主內容區
<div> 無語意的區塊，用來排版、組合模組；display 為 block 從新行開始
<p> 段落：主要用於承載文章、新聞或其他文本內容的段落；display 為 block 從新行開始
<section> 章節或功能區；display 為 block 從新行開始
<article> 可獨立發佈內容（部落格、新聞等）；display 為 block 從新行開始
<span>、<label>、<img>、<strong>、<em>(斜體)、<a>、<mark>、<code> 行內元件(inline-element)，無語意的區塊，用來修飾、強調重點；display 為 inline 不佔據整行
<footer> 頁尾區塊，放聯絡方式、社群連結、版權資訊

* 全域選擇器 * { margin: 0; padding: 0; } 畫面初始化
p 元件選擇器 p { color: blue; } 所有段落變藍

7. CSS Media Query

Media Query 用於響應式設計，根據不同裝置或螢幕尺寸調整樣式。

條件 意思 / 用途說明
min-width / max-width 最小／最大螢幕寬度（最常用，控制 RWD 斷點）
min-height / max-height 最小／最大螢幕高度（較少用，針對特殊裝置或全螢幕）

只在螢幕寬度小於等於767px時生效

@media (max-width: 767px) {
            .col{
                width: 100%;
            }
            .col h3{
                text-align: center;
                font-size:  24px;
            }
        }

針對平板電腦版型設計樣式：只在視窗寬度768-1076px時生效

@media (min-width: 768px) and (max-width: 1076px) {
            .col{
                width: calc(100% / 2);
            }
            .col h3{
                text-align: center;
                font-size:  24px;
            }
        }

8. CSS Position 定位

三種常見 CSS 定位方式的簡要解說：

1. position: static;

預設值，所有元素預設都是 static。

不會受到 top、right、bottom、left 屬性的影響。

元素會依照正常的文件流排列。

2. position: relative;

元素相對於 自己原本的位置進行偏移 ，元素仍然佔據原本的空間。

可以搭配 top、right、bottom、left 屬性微調位置。

3. position: absolute;

會尋找 上一層relative 當參考點。

可以用 top、right、bottom、left 精確控制位置。
