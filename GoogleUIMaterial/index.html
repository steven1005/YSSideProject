1: Material 引入模板

Material Design 3 的官方網頁元件（Web Components）核心理念是提供高度原子化的元件（如按鈕、卡片、文字框）

但它並不包含像 Bootstrap 那種 col-6 或 row 的格線系統類別。

現代網頁開發大多使用 CSS Grid 或 Flexbox，這已經是瀏覽器標配，Material 不希望強迫你學習一套私有的格線語法。

核心特性 - Core Features

深入了解 Material 3 如何透過 Web Components 標準提供現代化的互動體驗。

1. 動態配色系統 (Dynamic Color)

這套算法不只是換主題，更是為了確保視覺的一致性：

  • 色調調色盤 (Tonal Palettes)：只需定義一個「種子顏色」，系統自動生成 13 個階梯色調，確保符合對比度標準。
  • 一致性：切換深色模式時，元件會自動對應正確色調，無需手動調整個別按鈕。

2. Shadow DOM 樣式封裝

基於 Lit 開發，解決了長期以來的樣式污染問題：

  • 樣式隔離：元件內部的 CSS 不會影響外部，外部樣式也無法輕易干擾內部。
  • 標準化：原生支援 Web Components 標準，可於 React、Vue 或純 HTML 環境執行。

3. 狀態層設計 (State Layers)

元件內建精細的互動回饋機制：

  • 四種狀態：內建 Hover (懸停)、Focus (焦點)、Pressed (按下)、Dragged (拖曳) 層級。
  • 視覺回饋：點擊自動觸發 Ripple (漣漪) 效果，元件本身即包含所有必要的互動 JS。

4. 輔助功能與語義化 (A11y)

官方元件在設計之初即考慮到無障礙存取：

  • 螢幕閱讀器友善：內置豐富的 ARIA 標籤與語義化結構。
  • 鍵盤導航：完整支援鍵盤 Tab 切換與操作，符合大型企業開發規範。

5. 排版建議：Material Web vs Bootstrap 5

特性 Material Web Components Bootstrap 5
角色 元件（組件）本身 佈局（格線）與基礎樣式
樣式控制 CSS Variables (自定義屬性) Utility Classes (類別名稱)
卡片類型 Elevated, Filled, Outlined 單一 Card 類別
主要邏輯 JavaScript Property (如 .selected) Data Attributes (如 data-bs-toggle)

function getNameValue() {
    const field = document.getElementById('name-field-3');
    const result = document.getElementById('name-result-3');
    const value = field.value.trim();

    if (!value) {
        alert('請輸入姓名！');
        return;
    }
    result.innerHTML = `你好，<b>${value}</b>！`;
}

4: 卡片 - Cards

Material Web Components 提供了「零件」，但「排版（Layout）」需要結合其他技術（如 Bootstrap）來實現。

  • Elevated Card (陰影卡片)：具有陰影，適合從背景中凸顯內容 md-elevated-card
  • Filled Card (填充卡片)：沒有陰影，有底色（通常與背景對比），視覺較為平整 md-filled-card
  • Outlined Card (輪廓卡片)：只有邊框，沒有陰影或底色，適合簡潔風格 md-outlined-card
HTML
<md-elevated-card>
  <div style="padding: 16px;">
    <h3>響應式卡片</h3>
    <p>這張卡片會根據螢幕寬度自動調整。</p>
  </div>
</md-elevated-card>

function getSliderAndSwitch() {
    const slider = document.getElementById('slider-9');
    const sw = document.getElementById('switch-9');
    const res = document.getElementById('slider-result-9');

    // Material Web 中，Switch 的狀態屬性是 .selected (boolean)
    // Slider 的數值是 .value (number)
    res.innerHTML = `
        Value: ${slider.value} 

        Status: ${sw.selected ? 'ON (開啟)' : 'OFF (關閉)'}
    `;
}
